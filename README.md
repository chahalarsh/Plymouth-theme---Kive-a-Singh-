# Plymouth theme-Kive a Singh!
This repository contains Plymouth theme based on "kive a singh?!" meme.

# How to apply?
Please make sure that you have plymouth installed.
Either download .zip file (recomended) or download "kive a singh" folder and extract if applicable and then place the folder under the following path-

```/usr/share/plymouth/themes```

Then run the following command.

```sudo plymouth-set-default-theme -R kive_a_singh```

And Done!

# Preview on your system without restarting
To Preview it on your system without restarting run the following command

```
sudo plymouthd
sudo plymouth --show-splash
```
Note - This will display the currently set plymouth theme. To exit switch text terminal by pressing 

``` Ctrl + alt + f2/f3/f4...```

And then Run -

```
sudo plymouth --quit
```
